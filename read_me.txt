# Netflix Clone

## Overview

This project is a clone of the Netflix homepage, created using HTML and CSS. It demonstrates the design and layout skills needed to replicate the look and feel of a popular streaming service's user interface.

## Features

- Responsive design that adapts to various screen sizes
- Attractive UI resembling Netflix
- Use of modern CSS techniques
- Organized project structure with a dedicated `assets` folder for images and videos

## Project Structure

```
NetflixClone/
│
├── index.html
├── styles.css
├── README.txt
└── assets/
    ├── images/
    └── videos/
```

## Assets

All important images and videos used in this project are stored in the `assets` folder. Inside this folder, you will find subfolders for images and videos, making it easy to manage and reference these assets in the HTML and CSS files.

## How to Run

1. Clone the repository or download the project files.
2. Open `index.html` in your web browser to view the Netflix clone homepage.
